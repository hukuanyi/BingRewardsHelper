# Bing Rewards Helper

一个Microsoft Edge浏览器插件，自动完成Bing搜索任务以获取Rewards积分。

主要是自己在用。
需要感谢【潘钜森】大佬的脚本，链接：https://scriptcat.org/zh-CN/script-show-page/3947


## 功能特性

- ✅ 自动执行Bing搜索任务
- ✅ 支持电脑和移动设备搜索
- ✅ 每日搜索次数限制，避免过度运行
- ✅ 可自定义搜索间隔
- ✅ 自动每日重置统计信息
- ✅ 美观的弹出界面
- ✅ 实时统计和通知

## 安装说明

### 准备图标文件

由于版权原因，本插件不包含图标文件。请按照以下步骤准备图标：

1. 创建一个名为 `icons` 的文件夹
2. 准备三个尺寸的图标文件（PNG格式）：
   - `icon16.png` - 16x16像素
   - `icon48.png` - 48x48像素
   - `icon128.png` - 128x128像素

3. 图标建议：
   - 使用Bing或Microsoft Rewards相关的logo
   - 确保图标清晰且易于识别
   - 可以使用在线图标生成工具

### 安装到Edge浏览器

1. 打开Microsoft Edge浏览器
2. 在地址栏输入 `edge://extensions`
3. 启用右上角的"开发人员模式"
4. 点击"加载解压缩的扩展"
5. 选择本插件所在的文件夹
6. 插件安装完成

## 使用说明

### 基本使用

1. 点击浏览器工具栏中的插件图标
2. 在弹出界面中点击"开始搜索"按钮
3. 插件将自动执行Bing搜索任务
4. 可以随时点击"停止搜索"暂停任务

### 设置说明

- **启用自动搜索**：是否启用自动搜索功能
- **启用每日限制**：是否限制每日搜索次数
- **自动每日重置**：是否在每天自动重置统计信息
- **启用通知**：是否显示搜索完成等通知

### 搜索限制设置

- **每日最大搜索次数**：每天最多执行多少次搜索（默认30次）
- **搜索间隔**：每次搜索之间的间隔时间（默认30秒）
- **电脑搜索上限**：电脑端搜索的最大次数（默认30次）
- **移动搜索上限**：移动端搜索的最大次数（默认20次）

## 功能特点

### 每日运行次数检测

插件内置了完善的每日运行次数检测机制：

1. **自动重置**：每天自动重置搜索计数
2. **智能限制**：达到设定的每日上限后自动停止
3. **分类统计**：分别统计电脑和移动设备搜索次数
4. **实时监控**：实时显示当前搜索进度

### 安全机制

- 模拟真实用户搜索行为
- 随机搜索词生成
- 可配置的搜索间隔
- 避免过度运行的保护机制

## 注意事项

⚠️ **重要提醒**：

1. 请合理使用，避免过度搜索
2. 建议每日搜索次数不超过50次
3. 搜索间隔不要设置太短（建议至少30秒）
4. 定期检查Microsoft Rewards账户状态
5. 如遇到问题，请停止使用并联系Microsoft支持

## 技术说明

### 核心功能

1. **自动搜索**：自动执行Bing搜索请求
2. **任务调度**：使用alarms API进行定时任务
3. **存储管理**：使用chrome.storage.local保存配置和状态
4. **通知系统**：使用notifications API发送提醒

### 主要API使用

- `chrome.storage` - 存储配置和统计信息
- `chrome.tabs` - 管理浏览器标签页
- `chrome.alarms` - 定时任务调度
- `chrome.notifications` - 系统通知
- `chrome.runtime` - 消息通信
